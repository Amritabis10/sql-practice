# Peak Online Time

**Difficulty:** Easy

## Problem Description

Table: user_activity

```
+-----------------+----------+
| Column Name     | Type     |
+-----------------+----------+
| start_timestamp | timestamp|
| end_timestamp   | timestamp|
| user_count      | bigint   |
| device_type     | text     |
| time_difference | double   |
+-----------------+----------+
```

- Each row represents aggregated activity for a device type in a time window.

## Task

For each device type, find the time period with the highest number of active users.

The output should contain `user_count`, `time_period`, and `device_type`, where `time_period` is a concatenation of `start_timestamp` and `end_timestamp` in the format `start_timestamp to end_timestamp`.

Return the result table in any order.
